Comment utiliser ce générateur

  1. Remplissez tous les champs du formulaire
  2. Téléchargez une photo et des logos si nécessaire
  3. Cliquez sur "Générer la carte" pour prévisualiser
  4. Utilisez le bouton "Retourner la carte" pour voir le verso
  5. Téléchargez la carte avec le bouton "Télécharger"
  6. Utilisez les paramètres avancés pour personnaliser l'apparence
